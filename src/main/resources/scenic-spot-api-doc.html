景点接口文档

获取所有景点信息

GET http://localhost:8081/api/scenic-spots

接口说明

该接口用于获取数据库中所有景点的详细信息，返回一个包含所有景点对象的JSON数组。

返回数据结构

返回一个ScenicSpot对象数组，每个对象包含以下字段：

id Integer 景点ID（主键，自增）
title String 景点名称
img String 景点图片URL
introduce String 景点介绍
address String 景点地址
tag String 景点标签（如：自然风光、历史文化等）
times String 开放时间

前端调用示例

1. 原生JavaScript (Fetch API)

fetch('http://localhost:8081/api/scenic-spots')
    .then(response => response.json())
    .then(scenicSpots => {
        console.log('景点数据:', scenicSpots);
        // 处理景点数据
    })
    .catch(error => {
        console.error('获取景点数据失败:', error);
    });

2. Axios

axios.get('http://localhost:8081/api/scenic-spots')
    .then(response => {
        const scenicSpots = response.data;
        console.log('景点数据:', scenicSpots);
        // 处理景点数据
    })
    .catch(error => {
        console.error('获取景点数据失败:', error);
    });

3. Vue.js

<template>
    <div class="scenic-spots">
        <div v-for="spot in scenicSpots" :key="spot.id" class="spot-item">
            <h3>{{ spot.title }}</h3>
            <img :src="spot.img" :alt="spot.title">
            <p>{{ spot.introduce }}</p>
            <p>地址: {{ spot.address }}</p>
            <p>标签: {{ spot.tag }}</p>
            <p>开放时间: {{ spot.times }}</p>
        </div>
    </div>
</template>

<script>
import axios from 'axios';

export default {
    data() {
        return {
            scenicSpots: []
        };
    },
    mounted() {
        this.fetchScenicSpots();
    },
    methods: {
        fetchScenicSpots() {
            axios.get('http://localhost:8081/api/scenic-spots')
                .then(response => {
                    this.scenicSpots = response.data;
                })
                .catch(error => {
                    console.error('获取景点数据失败:', error);
                });
        }
    }
};
</script>

4. React

import React, { useState, useEffect } from 'react';
import axios from 'axios';

function ScenicSpotList() {
    const [scenicSpots, setScenicSpots] = useState([]);
    const [error, setError] = useState(null);

    useEffect(() => {
        axios.get('http://localhost:8081/api/scenic-spots')
            .then(response => {
                setScenicSpots(response.data);
                setError(null);
            })
            .catch(err => {
                setError('获取景点数据失败: ' + err.message);
                setScenicSpots([]);
            });
    }, []);

    if (error) {
        return 
{error}
;
    }

    return (
        

            
景点列表

            

                {scenicSpots.map(spot => (
                    

                        
{spot.title}

                        
                        
{spot.address}

                        
{spot.tags}

                        
开放时间: {spot.times}

                        
{spot.introduce.substring(0, 100)}...

                    

                ))}
            

        

    );
}

export default ScenicSpotList;
